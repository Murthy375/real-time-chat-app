# Read-Time Chat App

A one-to-one real-time chat application built with authentication and message history.

## Core Features

- User authentication(JWT)
- 1-to-1 messaging
- Real-time communication via Socket.IO
- Chat history
- Simple UI(Daisy UI)

## Tech Stack

Frontend:
- React
- Daisy UI
- Tailwind CSS
- Socket.IO (client)

Backend:
- Node.js
- Express
- Socket.IO
- MongoDB
